# Shelter Testing Practice

## Overview

A Java project for practicing unit testing using **JUnit 5**, **AssertJ**, and **Mockito**. The application code is pre-built; this repository contains the completed test suite.

## Technologies

- Java 21
- Maven
- JUnit 5
- AssertJ
- Mockito

## Completed Tasks

- ✅ Task 1 – Unit tests for business logic
- ✅ Task 2 – Parameterized tests
- ✅ Task 3 – Exception testing
- ✅ Task 4 – Collection and sorting tests
- ✅ Task 5 – Nested tests and CSV/file I/O
- ✅ Task 6 – Mockito mocking and interaction verification
- ✅ Stretch – Report writer file output tests
- ✅ Task 8 – Code coverage analysis and additional branch coverage

## Topics Covered

- JUnit 5
- AssertJ
- Parameterized Tests
- Exception Testing
- Nested Tests
- Collection Assertions
- File I/O Testing
- Mockito
- Code Coverage

## Run Tests

```bash
mvn test
```

Or run the test suite (or **Run with Coverage**) directly from IntelliJ IDEA.
